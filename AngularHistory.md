Angualar is a framwork
by Google
It was first released in 2010 as AngularJS (version 1.x)
In 2016, Angular 2 was released, which was a complete rewrite of the framework

angular seperates

logic (TS file)
UI (HTML file)
and styling (CSS file)

This separation of concerns allows for better maintainability and scalability of applications built with Angular.

Angular working:
Angular uses a component-based architecture, where the application is divided into reusable components. Each component consists of three main parts: the template (HTML), the class (TypeScript), and the styles (CSS).
